# bsky_tui
Bluesky client for the terminal

![](./assets/splash.png)

## Installation

```bash
git clone https://github.com/ksk001100/bsky_tui
cd bsky_tui
cargo install --path .
```

or

```bash
cargo install bsky_tui
```

## Usage

```bash
bsky_tui --help
bsky_tui config #=> generate config file, please edit it
bsky_tui
```
