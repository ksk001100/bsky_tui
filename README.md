# bsky_tui
Bluesky client for the terminal

## Installation

```bash
git clone https://github.com/ksk001100/bsky_tui
cd bsky_tui
cargo install --path .
```

or

```bash
cargo install bsky_tui
```

## Usage

```bash
export BLUESKY_EMAIL=<your email>
export BLUESKY_PASSWORD=<your password>
bsky_tui
```
